📝 题目

DEF公司当前股价 £5，共 1000万股，计划分配 £500万（可选择股利或回购）。

  1. (a) 若选择现金股利，每股股利是多少？发放后股价是多少？
  2. (b) 若选择回购，能回购多少股？回购后每股价值是多少？
  3. (c) 两种方式下，持有100股的股东财富有何不同？
📖 点击查看答案与解析

(a) 现金股利方式：

每股股利 = £500万 ÷ 1000万股 = £0.50/股

发放后股价 = £5 − £0.50 = £4.50（除息效应）


(b) 股票回购方式：

回购股数 = £500万 ÷ £5 = 100万股

剩余股数 = 1000万 − 100万 = 900万股

公司剩余价值 = £5000万 − £500万 = £4500万

回购后每股价值 = £4500万 ÷ 900万 = £5.00（不变）


(c) 持有100股的股东财富对比：

股利方式：100股 × £4.50 + £50现金 = £450 + £50 = £500

回购方式：100股 × £5.00 = £500

✅ 两种方式财富完全相同，验证了 MM 股利无关定理。

📝 题目二

GHI公司股票当前股价 £2.50，即将发放 £0.25 股利，预期股利增长率 5%。

  1. (a) 计算除息后（ex-dividend）股价。
  2. (b) 用 Gordon Growth Model 计算股权成本（Ke），注意应使用除息后股价作为 P₀。
  3. (c) 若股利改为 £0.40（其他条件不变），Ke 会如何变化？说明原因。
📖 点击查看答案与解析

(a) 除息后股价：

Ex-dividend price = £2.50 − £0.25 = £2.25


(b) 股权成本 Ke（Gordon Growth Model）：

公式：Ke = D₁ / P₀ + g

D₁ = £0.25 × (1 + 5%) = £0.2625（下一期股利）

P₀ = 除息后股价 = £2.25

Ke = £0.2625 / £2.25 + 5% = 11.67% + 5% = 16.67%

⚠️ 陷阱：必须用除息后股价作为 P₀，而非含息股价 £2.50。用 £2.50 会得到错误的 Ke = 10.5% + 5% = 15.5%。


(c) 股利增至 £0.40 时：

Ex-dividend price = £2.50 − £0.40 = £2.10

D₁ = £0.40 × 1.05 = £0.42

Ke = £0.42 / £2.10 + 5% = 20% + 5% = 25%

✅ Ke 上升。原因：更高的股利导致除息后股价更低，分子（D₁）增大而分母（P₀）减小，Ke 显著上升。这体现了股利政策与股权成本之间的权衡。
