⏰ Part 1 — Time Value of Money

📚 Why Does Time Matter? 为什么时间有价值？

费曼类比

今天给你£1,000，你可以存银行，一年后变成£1,050（5%利率）。所以"今天的£1,000"和"一年后的£1,000"不是同一回事——今天的更值钱。这就是时间价值的全部直觉。

Future Value 终值
FV = PV × (1 + r)ⁿ
今天的钱 → 未来值多少
Present Value 现值
PV = FV ÷ (1 + r)ⁿ
未来的钱 → 今天值多少
🔢 Lecture Example

Tuition Payment Problem

需要在4年后支付$7,000学费，银行利率3%。现在需要存多少？

PV = $7,000 ÷ (1.03)⁴ = $7,000 ÷ 1.1255 = $6,219.41

折现因子 = 1/(1.03)⁴ = 0.8885。今天存$6,219.41，4年后正好变成$7,000。

Fisher Effect: Nominal vs Real Rate 名义利率 vs 实际利率

Fisher Equation
(1 + nominal) = (1 + real) × (1 + inflation)
近似：nominal ≈ real + inflation
🍕 Pizza Example

Real Interest = Compensation for Delayed Consumption

名义利率5%，通胀3%。实际利率 = 1.05/1.03 − 1 = 1.94%。意思是：你延迟消费一年，能多买1.94%的披萨。实际利率 = 延迟消费的真实补偿。

💰 Part 2 — NPV: The Gold Standard

📚 Net Present Value 净现值

核心定义

NPV = 所有未来现金流的现值之和 − 初始投资

NPV > 0 → 接受（创造财富）｜NPV < 0 → 拒绝（销毁财富）｜NPV = 0 → 无差异

NPV Formula
NPV = −C₀ + C₁/(1+r) + C₂/(1+r)² + ... + Cₙ/(1+r)ⁿ
C₀ = 初始投资（负数）｜Cₙ = 第n年现金流｜r = 折现率

Annuity Shortcut 年金快捷计算

年金 = 等额、连续、期末的现金流

PV of Annuity = C × AF(r, n)

AF(r,n) = [1 − 1/(1+r)ⁿ] ÷ r　（年金因子，可查表）

永续年金（Perpetuity）：PV = C ÷ r　（无限期等额现金流）

⚠️ Q3 陷阱 — 年金表 vs 折现因子表

年金表（Annuity Table）的数字 = 未来n年每年£1的现值之和。折现因子表（Discount Factor Table）= 第n年£1的现值。两者完全不同，不能混用。

✅ 年金表：5年10% = 3.791，意思是未来5年每年£1，今天值£3.791
⚠️ Q4 陷阱 — 折现率变化对 NPV 和 IRR 的影响

折现率上升 → NPV 下降（未来现金流折现更多）。IRR 本身不变（它是项目固有属性，不受折现率影响）。

✅ 折现率↑ → NPV↓，IRR 不变。这是高频考点。
🏢 Real Case — Amazon AWS NPV Decision

How Amazon Justified $Billions in Cloud Infrastructure

Amazon 投资 AWS 自动化合同处理平台时，CFO 团队做了 NPV 分析：初始投资巨大（负现金流），但未来5年节省80%人工成本（正现金流）。NPV > 0，项目获批。这就是 NPV 在真实世界的用法——不是"感觉值得"，而是数字说话。

📈 Part 3 — IRR & Other Methods

📚 Internal Rate of Return 内部收益率

定义

IRR = 使 NPV = 0 的折现率。如果 IRR > 资本成本（hurdle rate），接受项目。

Interpolation Formula 插值法

IRR Approximation
IRR ≈ r₁ + [NPV₁ ÷ (NPV₁ − NPV₂)] × (r₂ − r₁)
r₁ = 低折现率（NPV为正）｜r₂ = 高折现率（NPV为负）
🔢 Q7 — Sienna plc IRR Calculation

插值法实例

已知：折现率20%时 NPV = +£200，折现率30%时 NPV = −£100。

IRR ≈ 20% + [200 ÷ (200+100)] × (30%−20%) = 20% + 0.667 × 10% = 26.7%

⚠️ IRR 的致命缺陷 — Q6 & Q9

再投资假设问题（Q6）：IRR 隐含假设项目释放的现金流可以按 IRR 本身再投资——这通常不现实。NPV 假设按资本成本再投资，更合理。

变动折现率问题（Q9）：如果每年的资本成本不同，IRR 无法处理（只有一个 IRR，无法与多个折现率比较）。NPV 可以逐年用不同折现率。

✅ IRR 不总是促进财富最大化。NPV 才是唯一直接对应股东财富的方法（Q10）。

📚 Payback & ARR — 为什么它们不够好

Payback Period 回收期

多少年收回初始投资。简单直观，但忽略时间价值，忽略回收期后的现金流。

ARR 会计收益率

平均利润 ÷ 平均投资。基于会计利润而非现金流，受折旧方法影响，不可靠。

⚠️ Q8 — 实际调查结论

调查显示：企业倾向于同时使用多种方法（NPV + IRR + Payback），而不是只用一种。IRR 和 Payback 的使用并没有减少。

✅ Q8 答案：A tendency for businesses to use more than one method.
