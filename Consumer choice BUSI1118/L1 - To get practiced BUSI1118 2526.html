Problem Set 1: Basic Utility Concepts

Exercise 1.1: Consider the utility function U(X, Y) = XY. Calculate the utility derived from the following consumption bundles:
  1. (X=2, Y=3)
  2. (X=4, Y=2)
  3. (X=5, Y=1)
  4. (X=0, Y=10)

Which bundle gives the highest utility? Explain your answer.

Solution:
Bundle Calculation Utility
(2, 3) U = 2 × 3 6
(4, 2) U = 4 × 2 8
(5, 1) U = 5 × 1 5
(0, 10) U = 0 × 10 0

Bundle (4, 2) gives the highest utility of 8. This demonstrates that for this utility function, maximizing utility involves balancing the two goods.

Exercise 1.2: Based on the utility function U(X, Y) = XY from the previous exercise:
  1. Calculate the marginal utility of good X (MUX) at point (4, 2)
  2. Calculate the marginal utility of good Y (MUY) at point (4, 2)
  3. Calculate the marginal rate of substitution (MRSXY) at point (4, 2)
  4. Interpret the meaning of the MRS value you calculated.
Solution:

For U(X, Y) = XY:

MUX = ∂U/∂X = Y

MUY = ∂U/∂Y = X

MRSXY = MUX/MUY = Y/X


At point (4, 2):

  1. MUX = Y = 2
  2. MUY = X = 4
  3. MRSXY = Y/X = 2/4 = 0.5
  4. Interpretation: At point (4, 2), the consumer is willing to give up 0.5 units of Y in exchange for 1 additional unit of X, while maintaining the same level of utility.

Solution to Exercise 1.3:

For U(X, Y) = X2 + Y2:

MUX = ∂U/∂X = 2X

Answer to d): The marginal rate of substitution varies with the consumption bundle because MRS = X/Y depends on both X and Y. As X increases relative to Y, MRS increases.

Exercise 2.1: A consumer has income I = $100 to spend on two goods: pizza (Ppizza = $10) and soda (Psoda = $2).
  1. Write the budget constraint equation.
  2. Find the maximum quantity of pizza the consumer can buy (if only pizza is purchased).
  3. Find the maximum quantity of soda the consumer can buy (if only soda is purchased).
  4. What is the slope of the budget line?
  5. Draw the budget line.

Solution to Exercise 2.1:

a) Budget constraint: 100 = 10Qpizza + 2Qsoda

b) Maximum pizza (Qsoda = 0): 100 = 10Qpizza + 2(0) → Qpizza = 10

c) Maximum soda (Qpizza = 0): 100 = 10(0) + 2Qsoda → Qsoda = 50

d) Slope = -Ppizza/Psoda = -10/2 = -5

Exercise 2.2: Using the same budget constraint from Exercise 2.1 (I = 100, Ppizza = 10, Psoda = 2):
  1. If the consumer buys 4 pizzas, how many sodas can they buy?
  2. If the consumer buys 6 pizzas, how many sodas can they buy?
  3. If the consumer buys 30 sodas, how many pizzas can they buy?
  4. If the consumer's income increases to $200, what happens to the budget line?
Solution to Exercise 2.2:

Budget constraint: 100 = 10P + 2S, where P = pizza quantity, S = soda quantity

a) If P = 4: 100 = 10(4) + 2S = 40 + 2S → 2S = 60 → S = 30

b) If P = 6: 100 = 10(6) + 2S = 60 + 2S → 2S = 40 → S = 20

c) If S = 30: 100 = 10P + 2(30) = 10P + 60 → 10P = 40 → P = 4

d) If income doubles to $200: The budget line shifts outward parallel to itself. Both intercepts double: maximum pizza = 20, maximum soda = 100. The slope remains unchanged at -5.

Key Point: When income changes, the budget line shifts parallel (same slope) because prices remain the same.

Problem Set 3: Consumer Equilibrium

Exercise 3.1: A consumer has utility function U(X, Y) = XY with income I = $100. The prices are PX = $5 and PY = $5.
  1. Find the marginal utilities MUX and MUY.
  2. Write the optimality condition (MUX/PX = MUY/PY).
  3. Find the optimal quantities of X and Y.
  4. Calculate the maximum utility at the optimum.
Page 5
Solution to Exercise 3.1:

For U(X, Y) = XY:

MUX = ∂U/∂X = Y

MUY = ∂U/∂Y = X


b) Optimality condition: Y/5 = X/5

Y = X


c) Budget constraint: 100 = 5X + 5Y

Substitute Y = X: 100 = 5X + 5X = 10X

X = 10, Y = 10


d) Maximum utility: U = X × Y = 10 × 10 = 100

Verification: At optimum (10, 10):
MUX/PX = 10/5 = 2
MUY/PY = 10/5 = 2
Since MU/P is equal across goods, consumer is at equilibrium. ✓
Exercise 3.2: Consider a consumer with U(X, Y) = X0.5Y0.5, I = $200, PX = $4, PY = $4.
  1. Set up the Lagrangian.
  2. Write down the first-order conditions.
  3. Solve for the optimal bundle.
  4. Find the Lagrange multiplier λ and interpret its meaning.
Page 6
Solution to Exercise 3.2:

a) Lagrangian: ℒ = X0.5Y0.5 + λ(200 - 4X - 4Y)

b) First-order conditions:

∂ℒ/∂X = 0.5X-0.5Y0.5 - 4λ = 0
∂ℒ/∂Y = 0.5X0.5Y-0.5 - 4λ = 0
∂ℒ/∂λ = 200 - 4X - 4Y = 0

c) From first two FOCs: 0.5X-0.5Y0.5 = 0.5X0.5Y-0.5

This implies: X-1Y = XY-1

Y = X

From budget: 200 = 4X + 4Y = 4X + 4X = 8X

X = 25, Y = 25


d) Lagrange multiplier:

From ∂ℒ/∂X: 4λ = 0.5(25)-0.5(25)0.5 = 0.5 × 0.2 × 5 = 0.5

λ = 0.5/4 = 0.125

Interpretation: The marginal utility of income is 0.125 utils per dollar. An additional $1 of income would increase maximum utility by approximately 0.125 utils.

Problem Set 4: Indifference Curves

Exercise 4.1: For the utility function U(X, Y) = XY:
  1. Derive the equation of the indifference curve for U = 25.
  2. Calculate the MRS at point (5, 5) and at point (10, 2.5).
  3. Explain the pattern in MRS as you move along the indifference curve.
Page 7
Solution to Exercise 4.1:

a) Indifference curve for U = 25: XY = 25 → Y = 25/X

This is a hyperbola.


b) MRS calculation:

For U = XY: MRS = Y/X

At point (5, 5): MRS = 5/5 = 1

At point (10, 2.5): MRS = 2.5/10 = 0.25


c) Pattern in MRS:

As you move rightward along the indifference curve (more X, less Y), the MRS decreases. This demonstrates diminishing marginal rate of substitution. When you have more X, you value additional X less and Y more, so you're less willing to trade Y for X.

Exercise 4.2: Sketch indifference curves for the following utility functions:
  1. U(X, Y) = 2X + Y (perfect substitutes)
  2. U(X, Y) = min{X, 2Y} (perfect complements)

For each, indicate the shape and calculate the MRS (if applicable).

Solution to Exercise 4.2:

a) U = 2X + Y (Perfect Substitutes):

  • Shape: Straight, parallel lines
  • MRS = MUX/MUY = 2/1 = 2 (constant)
  • Consumer is always willing to trade 2 units of Y for 1 unit of X

b) U = min{X, 2Y} (Perfect Complements):

  • Shape: L-shaped curves
  • MRS is undefined at the kink and 0 or ∞ elsewhere
  • Optimal consumption always at the kink where X = 2Y

Problem Set 5: Income and Price Effects

Exercise 5.1: Consider a consumer with U(X, Y) = XY, I = 100, PX = 10, PY = 10.
  1. Find the initial optimal bundle.
  2. If PX decreases to 5, find the new optimal bundle.
  3. Calculate the total effect, substitution effect, and income effect for good X.
Solution to Exercise 5.1:

For U = XY, MUX = Y, MUY = X


a) Initial (PX = PY = 10):

Optimality: Y/X = 10/10 = 1 → Y = X

Budget: 100 = 10X + 10Y = 10X + 10X = 20X

X = 5, Y = 5, U = 25


b) New (PX = 5, PY = 10):

Optimality: Y/X = 5/10 = 0.5 → 2Y = X

Budget: 100 = 5X + 10Y = 5(2Y) + 10Y = 20Y

Y = 5, X = 10, U = 50


c) Effects for good X:

  • Total effect: X increases from 5 to 10 (ΔX = +5)
  • Substitution effect: Find bundle at U = 25 with new prices:
    At U = 25, XY = 25 → Y = 25/X
    MRS = Y/X = 5/10 = 0.5
    New optimality condition (same): Y/X = 0.5 → 2Y = X
    From budget: 100 = 5X + 10Y = 5(2Y) + 10Y = 20Y
    Y = 5, X = 10 (but this is the new optimum, not the SE!)
    SE: For proper SE, we need the compensated demand. At the same utility (U = 25):
    Y/X = PX/PY = 5/10 = 0.5 → Y = 0.5X
    U = X(0.5X) = 0.5X2 = 25
    X = √50 ≈ 7.07, Y = 3.54
    SE = 7.07 - 5 = 2.07
  • Income effect: IE = Total - SE = 5 - 2.07 = 2.93
Page 9
Interpretation: For good X (normal good in this case):
  • Substitution effect (2.07): X increases because X became relatively cheaper
  • Income effect (2.93): X increases because consumer effectively became richer
  • Total effect (5): Sum of SE + IE
  • Both effects work in the same direction because X is a normal good
Exercise 5.2: Repeat Exercise 5.1 assuming good X is inferior.
  1. How would the income effect differ?
  2. What would be the total effect on quantity of X?
  3. Could this result in a Giffen good scenario? Explain.
Solution to Exercise 5.2:

If good X were inferior:

a) The income effect would work against the substitution effect. When PX decreases, the SE would increase X (as before), but the IE would decrease X because the consumer is now effectively richer and buys less inferior goods.

b) Total effect = SE + IE. If |IE| > |SE|, the total effect could be a decrease in X despite the price decrease.

c) Yes, this could result in a Giffen good scenario. If the income effect (decrease in X) is strong enough to overcome the substitution effect (increase in X), the total effect would show X decreasing as PX falls, violating the law of demand.

Note: This is the theoretical case of Giffen goods, which are extremely rare in reality.

Problem Set 6: Cobb-Douglas Preferences

Exercise 6.1: For U(X, Y) = X0.3Y0.7 with I = 120, PX = 6, PY = 4:
  1. What share of income is spent on good X?
  2. What share of income is spent on good Y?
  3. Find the optimal quantities of X and Y.
  4. Calculate the total expenditure at the optimum.
Page 10
Solution to Exercise 6.1:

For Cobb-Douglas utility U = XαYβ:

Key property: Consumer spends constant share of income on each good:

  • Share on X = α/(α + β)
  • Share on Y = β/(α + β)

Here: α = 0.3, β = 0.7


a) Share on X = 0.3/(0.3 + 0.7) = 0.3/1 = 30%

b) Share on Y = 0.7/(0.3 + 0.7) = 0.7/1 = 70%

c) Optimal quantities:

X = 0.3 × (I/PX) = 0.3 × (120/6) = 0.3 × 20 = 6
Y = 0.7 × (I/PY) = 0.7 × (120/4) = 0.7 × 30 = 21

Optimal bundle: (6, 21)

d) Total expenditure: 6(6) + 4(21) = 36 + 84 = 120 = I ✓

Expenditure on X: 36/120 = 30% ✓

Expenditure on Y: 84/120 = 70% ✓

Exercise 6.2: For the same utility function from Exercise 6.1:
  1. What happens to the optimal quantities if income doubles to I = 240?
  2. What happens to the optimal quantities if both prices double (PX = 12, PY = 8)?
  3. What happens to the optimal quantities if income and all prices double?
Solution to Exercise 6.2:

a) If income doubles to 240:

X = 0.3 × (240/6) = 0.3 × 40 = 12

Y = 0.7 × (240/4) = 0.7 × 60 = 42

Both quantities double (homogeneous of degree 1).


b) If prices double (I = 120):

X = 0.3 × (120/12) = 0.3 × 10 = 3

Y = 0.7 × (120/8) = 0.7 × 15 = 10.5

Both quantities halve.


c) If income and all prices double (I = 240, PX = 12, PY = 8):

X = 0.3 × (240/12) = 0.3 × 20 = 6

Y = 0.7 × (240/8) = 0.7 × 30 = 21

Both quantities unchanged (zero homogeneity for prices, unit homogeneity for income).

Problem Set 7: Comparative Statics

Exercise 7.1: Consider two consumers with different utility functions facing the same prices and income:

Consumer A: UA(X, Y) = XY
Consumer B: UB(X, Y) = X2Y2
I = 100, PX = 5, PY = 5
  1. Find Consumer A's optimal bundle.
  2. Find Consumer B's optimal bundle.
  3. Compare the utility levels at each consumer's optimum.
  4. Who has higher utility? Explain why.
Solution to Exercise 7.1:

Consumer A (U = XY, PX = PY = 5):

Optimality: Y/X = 1 → Y = X

Budget: 100 = 5X + 5Y = 10X

X = 10, Y = 10, UA = 100


Consumer B (U = X2Y2, PX = PY = 5):

MUX = 2X, MUY = 2Y

Optimality: 2X/5 = 2Y/5 → X = Y

Budget: 100 = 5X + 5Y = 10X

X = 10, Y = 10, UB = 1002 = 10,000


c) Comparison: UB = 10,000 > UA = 100

d) Consumer B has much higher utility. This is because utility functions are not comparable across different functional forms—utility is ordinal, not cardinal. We cannot conclude that B is "better off" than A just because UB > UA. The important question is whether each consumer maximizes their own utility given their constraints.

Warning: Never compare utility values across different consumers directly! Utility is a tool for modeling preferences, not a measurable quantity of satisfaction.

Exercise 7.2: For a single consumer with U(X, Y) = 2X + Y (perfect substitutes), I = 60, PX = 3, PY = 2:
  1. Calculate MUX/PX and MUY/PY.
  2. What is the optimal bundle?
  3. What type of solution is this?
Solution to Exercise 7.2:

For U = 2X + Y:

MUX = 2, MUY = 1

a) MUX/PX = 2/3 ≈ 0.67

MUY/PY = 1/2 = 0.5


b) Since MUX/PX > MUY/PY, the consumer gets more utility per dollar from good X and should buy only X.

Optimal bundle: X = 60/3 = 20, Y = 0

c) This is a corner solution because the consumer buys zero of good Y.

Key Point: For perfect substitutes, consumers typically specialize in the good with higher marginal utility per dollar, leading to corner solutions.
Exercise 7.3: For a consumer with U(X, Y) = min{2X, Y} (perfect complements), I = 80, PX = 4, PY = 2:
  1. What is the optimal consumption ratio?
  2. Find the optimal bundle.
  3. What type of solution is this?
Page 13
Solution to Exercise 7.3:

For U = min{2X, Y}, at the kink (optimum): 2X = Y

a) The optimal consumption ratio is X:Y = 1:2 (or equivalently, Y = 2X)


b) Budget constraint: 80 = 4X + 2Y

Substitute Y = 2X: 80 = 4X + 2(2X) = 8X

X = 10, Y = 20

Optimal bundle: (10, 20)

Utility: U = min{2(10), 20} = min{20, 20} = 20


c) This is a corner solution (specifically, a kink solution) because optimal consumption occurs at the kink of the L-shaped indifference curve where 2X = Y.

Note: For perfect complements, consumers always consume goods in the fixed ratio defined by the utility function coefficients.

Problem Set 8: Slutsky Equation

Exercise 8.1: Consider the utility function U(X, Y) = X0.5Y0.5 with I = 200, PX = 4, PY = 5.
  1. Derive the demand functions for X and Y.
  2. Verify that the demand functions are homogenous of degree 0 in prices and homogeneous of degree 1 in income.
Solution to Exercise 8.1:

For Cobb-Douglas U = XαYβ with α = β = 0.5:

Demand functions:

X = (α/(α+β))(I/PX) = 0.5 × (200/4) = 0.5 × 50 = 25
Y = (β/(α+β))(I/PY) = 0.5 × (200/5) = 0.5 × 40 = 20

So X = 25/PX, Y = 20/PY


b) Homogeneity check:

  • If prices double (PX = 8, PY = 10): X = 25/8, Y = 20/10 (both halve, homogeneity degree 0 in prices)
  • If income doubles (I = 400): X = 100/4 = 25, Y = 80/5 = 16 (both double, homogeneity degree 1 in income)
  • If both double: X = 50/4 = 12.5, Y = 40/5 = 8 (same as original)

Exercise 8.2: Using the demand functions from Exercise 8.1:
  1. Calculate the own-price elasticity of demand for good X.
  2. Calculate the cross-price elasticity of demand for good X with respect to PY.
  3. Calculate the income elasticity of demand for good X.
Solution to Exercise 8.2:

Demand function for X: X = 25/PX

At PX = 4: X = 25/4 = 6.25


a) Own-price elasticity EXX:

EXX = (dX/dPX) × (PX/X) = (-25/PX2) × (PX/X)

For X = 25/PX: dX/dPX = -25/PX2

EXX = (-25/PX2) × PX2/25 = -25/25 = -1

Unit elastic demand for Cobb-Douglas utility.


b) Cross-price elasticity EXY:

Demand for X does not depend on PY, so dX/dPY = 0

EXY = 0

Goods X and Y are neither substitutes nor complements (independent).


c) Income elasticity EXI:

EXI = (dX/dI) × (I/X) = (1/PX) × (I/X)

At X = 25/PX: EXI = (1/4) × (4/(25/PX)) = (1/4) × (4PX/25) = PX/25 = (1/PX)(PX/25)

At PX = 4: EXI = 4/25 = 0.16

Since 0 < EI < 1, good X is a necessity.

Problem Set 9: Applications

Exercise 9.1: A government imposes a tax of $2 per unit on good X. Before the tax, PX = $5, PY = $5, I = $100.
After the tax, PX = $7 (tax-inclusive).
For U(X, Y) = X0.5Y0.5:
  1. Find the optimal bundle before tax.
  2. Find the optimal bundle after tax.
  3. Calculate the tax revenue collected.
  4. Calculate the excess burden (deadweight loss) of the tax.
Page 15
Solution to Exercise 9.1:

For U = X0.5Y0.5 (α = β = 0.5):


a) Before tax (PX = PY = 5, I = 100):

X = Y = 0.5 × 100/5 = 10

Optimal: (10, 10), U = 100


b) After tax (PX = 7, PY = 5, I = 100):

Optimality: Y/X = 7/5 = 1.4

Y = 1.4X

Budget: 100 = 7X + 5(1.4X) = 7X + 7X = 14X

X = 7.14, Y = 10

New optimal: (7.14, 10), U = 71.4


c) Tax revenue = tax × quantity after tax = 2 × 7.14 = $14.28

d) Excess burden (deadweight loss):

Consumer surplus lost = Change in utility + Tax revenue - Cost of tax payment

This is an advanced concept - for this course, focus on the change in quantities and utility.

Utility loss = 100 - 71.4 = 28.6

Key Point: Taxes create a wedge between what consumers pay and what producers receive (in this simplified model). The consumer reduces consumption of the taxed good, moving to a lower utility level.
